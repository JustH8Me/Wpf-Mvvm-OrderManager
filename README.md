# Order CRM System (WPF + MVVM)

CRM-система для управления заказами сервисного центра.
Проект выполнен для демонстрации навыков работы с архитектурой MVVM и базой данных.

##  Технологии
*   **Язык:** C# (.NET 9)
*   **UI:** WPF (Windows Presentation Foundation)
*   **Архитектура:** MVVM (Model-View-ViewModel)
*   **База данных:** SQLite + Entity Framework Core
*   **DI:** Microsoft.Extensions.DependencyInjection

##  Функционал
1.  **Дашборд:** Статистика по доходам и заказам с фильтрацией по датам.
2.  **Активные заказы:** Создание, перевод в работу, завершение.
3.  **История:** Просмотр завершенных заказов (Read-Only).
4.  **Умный поиск:** Поиск по ID, Имени клиента или Описанию.
